# ComCall

## PL

### O projekcie
<br>
Projekt dotyczy platformy internetowej do prowadzenia różnorodnych kursów. Platforma ma różne funkcjonalności w zależności od użytkownika:  administrator, nauczyciel, kursant. Dzięki platformie możliwe jest prowadzenia kursów online w czasie rzeczywistym, ale także komunikowanie się z użytkownikami poprzez funkcje czatu. W celach edukacyjnych w platformie stworzono specjalnie miejsce, gdzie nauczyciel może wystawić kursantom zadanie a oni je załadować. Dodatkowo istnieje możliwość ustawiania dat oddania zadania, czy też planowania spotkań za pomocą funkcji kalendarza. Platforma posiada niezbędne funkcje umożliwiające prowadzenie szkoleń online.

## ENG

### About the project 
<br>
The project concerns an internet platform for conducting various courses. The platform has different functionalities depending on the user: administrator, teacher, student. Thanks to the platform, it is possible to conduct online courses in real time, but also to communicate with users through the chat function. For educational purposes, a special place has been created on the platform, where the teacher can give the students a task and they can upload it. Additionally, it is possible to set deadlines for the task submission or schedule meetings using the calendar function. The platform has the necessary functions to conduct online training.
